## Highlights

This release sets the [minimatch](https://www.npmjs.com/package/minimatch) dependency version used in ESLint to `^3.1.5`. This change avoids a bug in a previous minimatch release that could cause ESLint to not recognize certain files. A transitive dependency on minimatch was also updated to `^3.1.5` to include a fix for a recently published security issue.
