Buttons

There are three types of buttons: primary, secondary, and "ghost". To render the proper element, a `behavior` key is provided, which takes either a `action` or a `link` value. `link` is the default, which will render an <a> tag that looks like a button. The `action` behavior implies that it is a button _that performs an action_ and is therefore rendered as a `<button type="button">`.
{{ button({ behavior: "action", type: "primary" }) }} {{ button({ behavior: "action", text: "I perform an action", type: "secondary" }) }} {{ button({ behavior: "action", text: "I perform an action", type: "ghost" }) }}
